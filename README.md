# JWT gRPC Interceptors

![License](https://img.shields.io/dub/l/vibe-d.svg)

## Installation

Use go get.

```shell
go get github.com/go-funcards/jwt-grpc
```

Then import **jwt-grpc** package into your own code.

```go
import "github.com/go-funcards/jwt-grpc"
```
